STEPS
1. Unzip CS4354_HW3.zip
2. Look for simpleLib folder
3. Open Eclipse IDE
4. File > Import... > General > Existing Projects into Workspace
5. Select Next
6. Under "Select root directory" select Browse...
7. Browse for simpleLib folder.

8. Run the app
9. Add some users.
10. Add some documents.
11. Then you can search, or borrow and return documents.
